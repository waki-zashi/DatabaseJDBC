# Library Database Management System (Java + PostgreSQL)

## Описание проекта

Данный проект представляет собой настольное приложение для управления библиотечной базой данных.  
Программа разработана на языке **Java** с использованием **Swing** для графического интерфейса и **PostgreSQL** в качестве системы управления базами данных.

Приложение позволяет:

- подключаться к серверу PostgreSQL
- создавать и удалять базы данных
- выполнять автоматическую инициализацию структуры базы данных
- управлять пользователями и ролями
- выполнять операции с книгами (добавление, редактирование, поиск, удаление)
- вести журнал действий пользователей

Проект демонстрирует работу с **JDBC**, **хранимыми процедурами**, **функциями**, **ролями и правами доступа** в PostgreSQL.

---

## Основные возможности

### Управление базой данных
Программа позволяет:

- подключаться к серверу PostgreSQL
- создавать новую базу данных
- автоматически инициализировать структуру базы
- удалять существующие базы данных

При создании новой базы данных автоматически создаются:

- таблица книг
- роли пользователей
- функции и процедуры для работы с данными
- права доступа для различных ролей

---

### Управление книгами

Поддерживаются следующие операции:

- добавление новой книги
- редактирование информации о книге
- удаление книги по ID
- очистка всей таблицы
- просмотр всех книг

Также реализован поиск:

- по названию
- по автору
- по жанру

Все операции выполняются через **хранимые процедуры и функции PostgreSQL**.

---

### Управление пользователями

В системе используются роли:

**admin_role**

Администратор может:

- добавлять книги
- редактировать книги
- удалять книги
- очищать таблицу
- создавать новых пользователей

**guest_role**

Гость может:

- просматривать список книг
- выполнять поиск по книгам

Создание пользователей выполняется через хранимую процедуру:

```

sp_create_db_user(username, password, role)

```

---

### Логирование

Приложение ведёт журнал действий.

Логи отображаются в интерфейсе программы и сохраняются в файл.

Фиксируются следующие события:

- подключение к базе данных
- создание базы данных
- удаление базы данных
- вход пользователя
- выполнение операций с книгами
- ошибки базы данных

---

## Структура базы данных

### Таблица книг

```

books

```

Поля:

| Поле | Тип | Описание |
|-----|-----|----------|
| id | SERIAL | идентификатор книги |
| title | VARCHAR | название |
| author | VARCHAR | автор |
| genre | VARCHAR | жанр |
| year | INT | год издания |

---

### Хранимые процедуры

```

sp_add_book
sp_update_book
sp_delete_by_id
sp_clear_table
sp_create_db_user

```

Назначение:

- добавление книги
- обновление книги
- удаление книги
- очистка таблицы
- создание пользователя

---

### Хранимые функции

```

sp_get_all_books
sp_find_by_title
sp_find_by_author
sp_find_by_genre

```

Назначение:

- получение списка всех книг
- поиск книг по различным параметрам

---

## Архитектура приложения

Проект реализован по архитектурному принципу **MVC (Model-View-Controller)**.

**Model**

Работа с базой данных:

- JDBC подключение
- выполнение SQL-запросов
- вызов процедур и функций

**View**

Графический интерфейс пользователя:

- окна приложения (Swing)
- формы ввода
- таблицы отображения данных
- область логов

**Controller**

Логика приложения:

- обработка действий пользователя
- вызов операций базы данных
- обновление интерфейса

## Требования для запуска

Необходимо установить:

- Java 17 или выше
- PostgreSQL
- JDBC драйвер PostgreSQL

Параметры подключения по умолчанию:

```

Host: localhost
Port: 5433
Database: library

```

---

## Запуск приложения

1. Запустить PostgreSQL сервер
2. Запустить приложение
3. В окне Database Setup указать:

- host
- port
- database

4. Создать новую базу данных или подключиться к существующей
5. Войти в систему через окно авторизации
